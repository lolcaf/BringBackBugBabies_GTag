# Bring Back Bug Babies
This is my first ever gtag mod!

Ever missed doug the bug's babies?

I made a mod to bring them back with full **customizability!**

_I am new to mod making so if you find any bugs please report it to the Issues tab_

/BBBBIcon.png

# Controls

All controls are keyboard only, so it doesn't get in the way of your VR experience :)

The + key adds a new bug baby

The - key removes a bug baby

**There is no set limit to how many bug babies you can have!**

# Dependencies

This mod has no dependencies
